# ansible POWERLEVEL9K ZSH
Installs zsh with powerlevel 9k and iterm2 with nerdfont

## Quickstart
1. To test playbook locally:
  ```
  ansible-playbook test-role.yml
  ```
2. Swap the default shell by running:
  ```
  chsh -s /bin/zsh
  ```
